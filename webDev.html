RIVER OF STYX GAME

UI / UX | GRAPHIC DESIGN | REACT | CSS | JAVASCRIPT(JQUERY)


STYX is a four-person online game based off the Mensa Select Winner board game, Lanterns, and the ancient Greek myth of the River Styx. STYX is designed to be an educational tool for younger kids to practice spacial reasoning, gain insight into Greek culture, as well as increase critical thinking skills. All graphics were designed specially for this project.
